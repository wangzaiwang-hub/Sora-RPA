# 图标文件说明

## 需要的图标文件

请在此文件夹中放置以下图标文件：

1. **icon16.png** - 16x16 像素
   - 用于浏览器工具栏
   
2. **icon48.png** - 48x48 像素
   - 用于扩展管理页面
   
3. **icon128.png** - 128x128 像素
   - 用于 Chrome 网上应用店

## 图标设计建议

- 使用简洁的设计
- 主题色：紫色渐变 (#667eea → #764ba2)
- 图标内容：可以是任务列表、进度条、或者 Sora 相关的图形
- 背景：透明或白色
- 格式：PNG（支持透明度）

## 临时解决方案

如果暂时没有图标，可以：

1. 使用在线图标生成器创建简单图标
2. 从其他项目复制类似的图标
3. 使用纯色方块作为占位符

## 图标生成工具推荐

- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Canva](https://www.canva.com/)
- Photoshop / GIMP

## 注意事项

- 确保图标文件名完全匹配（区分大小写）
- 确保图标尺寸正确
- 建议使用 PNG 格式
- 保持图标风格一致
